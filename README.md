# Weather App

A simple and intuitive weather application that allows users to check current weather conditions for any city around the world.

## Features

- Real-time weather data using OpenWeatherMap API
- Display of current temperature in Celsius
- Humidity percentage indicator
- Wind speed information
- Dynamic weather icons based on current conditions
- Error handling for invalid city names
- Supports both button click and Enter key for searching

## Technologies Used

- HTML
- CSS
- JavaScript
- OpenWeatherMap API

## How to Use

1. Enter a city name in the search box
2. Either click the search button or press Enter
3. View the current weather information including:
   - Temperature
   - Humidity
   - Wind Speed
   - Weather condition icon

## Weather Conditions Supported

- Clear sky
- Cloudy
- Rain
- Drizzle
- Mist

## Note

This app requires an active internet connection to fetch weather data. Make sure you have internet connectivity while using the application.
